# YumDrop

**YumDrop** is a full-stack food delivery web application that lets users browse, order, and enjoy meals with ease. Itâ€™s built with a modern tech stack to ensure fast performance, scalability, and a smooth user experience.

## Tech Stack

- **Backend:** Spring Boot (Java)
- **Frontend:** React.js + Bootstrap
- **Database:** MongoDB
- **Cloud Storage:** AWS S3 (for food images)

## Features

- Browse a wide variety of foods
- Upload and store food images securely with AWS S3
- Add foods to your cart
- Place orders
- Search for specific dishes
- Fast and responsive UI
